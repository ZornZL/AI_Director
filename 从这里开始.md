# AiBianDao 从这里开始

AiBianDao 是面向新手的 Seedance 2.0 AI 视频编导系统。它把一个主题、大赛规则、产品命题或用户自定义想法，逐步变成可审核的完整脚本，以及可独立复制到 Seedance 2.0 的 4—15 秒逐镜任务。

> 本系统只规划 AI 生成视频。摄影、机位、灯光、表演、场景、演员和道具均指 Seedance 2.0 与配套图片资产的生成控制，不代表现实拍摄。

## 新手主流程

```text
主题/大赛规则/产品命题
→ 3个差异明显的创意方向
→ 用户锁定1个方向
→ 完整脚本草案
→ 逻辑初审与A/B/C/D/E确认
→ 唯一锁定文案
→ 平台/画幅/总片长/Seedance入口参数确认
→ 导演逐镜脚本
→ 核心人物、场景、产品和关键帧资产
→ 每镜独立的4—15秒Seedance任务
→ 先测身份、最难动作、产品/文字
→ 用户生成、验收和单变量重试
```

如果用户已经提供具体脚本，则从“逻辑初审”开始，不重复创意方向。

## 三个必须由用户确认的内容

1. 选择哪个创意方向。
2. 哪个脚本版本正式锁定。
3. 人物外观、画幅、关键资产和高风险任务是否可进入生成。

## 故事版怎么用

故事版是可选的看图预审，不是稳定生产能力，也不是强制步骤。

- 单镜头或简单短片：可以跳过。
- 多场景、超过6镜、跨年龄或连续性要求高：建议先做故事版。
- 故事版只检查剧情、构图、人物一致性和动作可读性；不得直接当作正式角色图、产品图或生产首帧。

## Seedance逐镜任务最低标准

每镜必须包含：4—15秒建议时长、画幅、参考资产、时间与场景、一个主要镜头意图、逐秒动作、声音/对白白名单、连续性、禁止项、验收标准和失败降级方案。

## 当前证据边界

- 唯一完整闭环案例：`06-视频制作项目/ABD-PROD-2026-002-houlai-mama/`。
- 当前验证的是“人工操作 Seedance + 系统编译和复盘”，不代表全自动操作网页、自动剪辑或一次生成必成。
- 正式生效经验只保留在 `ABD-CASE-2026-006` 与 `ABD-RISK-2026-002`。

## 第一次使用

1. 填写 `04-模板库/Seedance2简单文案输入模板.md`，不知道的字段可留空。
2. 从主题开始时调用 `create-video-concept`；有具体脚本时调用 `refine-video-script-logic`。
3. 文案锁定后调用 `compile-seedance2-shots`。
4. 先确认核心资产，再测试三个高风险任务；失败时一次只改一个变量。

详细入口见 `00-系统总控/总索引.md`。
