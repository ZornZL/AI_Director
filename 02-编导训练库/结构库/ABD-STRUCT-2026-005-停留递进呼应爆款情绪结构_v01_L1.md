# 停留递进呼应爆款情绪结构

## 等级

L1：由已完成项目《后来，我成了妈妈的妈妈》、参考片《五厘米》以及用户对 `ABD-PROD-2026-004-borrow-one-minute`、`ABD-PROD-2026-005-shao-yi-ju-hua` 的反馈共同提炼。尚未用新项目完成 L2 生产验证。

## 结构公式

```text
强停留钩子
→ 低门槛问题或异常关系
→ 第一层轻答案/轻冲突
→ 第二层现实处境
→ 第三层关系代价
→ 第四层遗憾或来不及
→ 回到开头同一物件/台词/动作
→ 主题深化并把问题交还给观众
```

## 核心判断

爆款情绪短片不是“哲理答案合集”，而是“观众愿意停下来的问题 + 逐层变重的故事 + 结尾重新理解开头”。

《后来，我成了妈妈的妈妈》成立的核心不是亲情题材本身，而是：

- 开头有认知错位：白发妈妈问成年女儿“你认识我女儿吗？”
- 中间有关系递进：女儿小时候推开妈妈，长大后错过妈妈，后来反向照顾妈妈。
- 有重复母题：牵手、过马路、不能松手、黄精果、无葱面。
- 结尾有动作回扣：女儿把手放回妈妈掌心，完成照顾关系反转。

《五厘米》成立的核心不是“五厘米”本身，而是：

- 问题低门槛，前段好笑。
- 每个答案都有具体可见落点。
- 后段不断把轻问题推向人生代价。
- 结尾让观众回看开头，发现玩笑背后是遗憾。

## 必备四件事

### 1. 开头必须有停留

停留来自以下至少一种：

- 认知错位：人物身份、关系或动作不合常规，但真实。
- 轻问题重答案：问题像玩笑，答案逐渐变深。
- 反常动作：一个角色在不该等的地方等、不该忘的事忘、不该说的话没说。
- 关系悬念：观众立刻想知道“他们到底是什么关系”。

不合格开头：

- 直接讲道理。
- 直接抒情旁白。
- 直接给一串正确答案。
- 只有标题好，画面和第一句没有悬念。

### 2. 中间必须递进，不只是并列

每一段必须比上一段多一个新维度：

- 从好笑到温暖。
- 从个人愿望到关系责任。
- 从当下选择到多年后果。
- 从可以补救到已经来不及。

不合格中段：

- 每个案例都很好，但彼此只是排队。
- 每个角色都说一句金句，却没有因果或情绪升级。
- 旁白负责解释深刻，画面只负责摆拍。

### 3. 情绪必须有故事性

故事性不等于复杂剧情。最小故事单元是：

```text
角色是谁
→ 他现在想要什么
→ 为什么这一刻难
→ 他选择或没选择什么
→ 这个选择留下什么余味
```

如果一个答案只有概念，没有“这一刻为什么难”，就不能入片。

### 4. 末尾必须深化主题

结尾不能只重复主题句，必须完成回扣：

- 回到开头同一动作，但人物关系变了。
- 回到同一句台词，但说话人变了。
- 回到同一物件，但意义变了。
- 回到同一问题，但观众已经听不成玩笑。

### 5. 台词必须精简但有深意

爆款情绪短片里的关键台词不是讲道理，而是把一个人没说出口的关系代价短短说出来。

合格台词必须满足：

- 有对象：知道这句话真正说给谁。
- 有代价：这句话不说、晚说或说错，会留下什么。
- 有画面：画面中有动作、物件、空间或停顿承接。
- 有余味：观众能从上下文悟出人生哲理，而不是角色直接说破。
- 足够短：能用半句话、停顿或一个具体称呼表达的，不写成长解释。

不合格台词：

- 直接讲“人生、亲情、遗憾、珍惜、成长”等抽象道理。
- 用一整段对白解释人物为什么难过。
- 每个角色都像作者分身，轮流输出正确观点。

## 文案硬门槛

进入分镜前必须满足：

1. 前 3-8 秒有明确停留钩子。
2. 前 20 秒能让观众知道“为什么要继续看”。
3. 中段不是并列清单，至少有 3 次情绪升级。
4. 每个案例都有可见动作、物件或空间，不只是一句抽象答案。
5. 最后三个段落明显比前三个更重。
6. 结尾必须回扣开头的台词、动作、物件或关系。
7. 旁白不得替代剧情推进。
8. 任何金句都必须来自人物处境，而不是作者总结。
9. 关键台词必须短、具体、有对象、有未说出口的代价，并且让哲理隐含在关系和画面里。

## 对《借你一分钟》的适用方式

适合采用：

```text
轻问题：如果能借你一分钟，你想放在哪里？
→ 前段轻答案：赖床、地铁门、下课铃
→ 中段现实答案：交卷前、红灯前、回答前、挂电话前
→ 后段关系答案：关门前、睡着前、开口前、语音后面
→ 首尾回扣：开头以为一分钟给自己，结尾发现想留给家人
```

风险：

- 如果只做 12 个答案，会像答案合集。
- 必须让至少一个人物在结尾发生认知变化，或让同一个动作回到开头。

## 对《少一句话》的适用方式

适合采用：

```text
轻问题/钩子：少说一句话，会改变什么？
→ 前段轻误会：忍住一句吐槽、少一句争辩
→ 中段关系裂缝：少一句关心、少一句道歉、少一句挽留
→ 后段不可逆：少一句告别、少一句“我到了”、少一句“我爱你”
→ 结尾回扣：最想补上的不是长篇大论，而是一句最普通的话
```

风险：

- 容易写成金句合集。
- 必须把“少一句话”落到具体场景和具体对象。

## 禁用

- 不得把“爆款”理解为强行催泪。
- 不得只堆老人、死亡、病房、遗物来制造重感。
- 不得照搬《后来妈妈》失智母女、牵手过马路、黄精果、无葱面完整链路。
- 不得照搬《五厘米》具体案例链。
