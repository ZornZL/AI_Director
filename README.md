# AI_Director

AI_Director 是一个面向新手的 Seedance 2.0 AI 视频编导与逐镜任务编译系统。

## 能做什么

```text
主题/大赛规则/产品命题
→ 3个创意方向
→ 用户选择
→ 完整脚本
→ 逻辑审核与锁定
→ 导演分镜
→ 人物/场景/产品/关键帧资产
→ 独立4—15秒Seedance任务
→ 人工生成、验收和重试
```

用户已有具体脚本时，可以直接从逻辑审核开始。系统不规划真人拍摄，也不承诺自动控制网页、自动剪辑、自动发布或一次生成必成。

## 当前验证程度

唯一完整闭环案例是 `06-视频制作项目/ABD-PROD-2026-002-houlai-mama/`。它验证了锁定文案、资产、逐镜生成、失败重试、剪辑成片、抽帧复盘、基础音画检测、规则入库和发布前检查。尚无发布数据，因此当前最高等级为 L2。

## 快速开始

```powershell
git clone https://github.com/ZornZL/AI_Director.git D:\AI_Director\AI_Director
```

在 Codex 中打开该目录，并发送：

```text
这是 AI_Director 项目的专职任务。
请先读取《从这里开始.md》和《00-系统总控/总索引.md》，
再按本次任务调用相关 Skill。
```

## 最小目录

```text
AGENTS.md                 项目执行规则
从这里开始.md            新手入口
00-系统总控/              流程、能力边界、看板和知识索引
01-品牌与产品库/          品牌与产品基础资料
02-编导训练库/            仅保留002验证的案例与失败规则
04-模板库/                输入、审核、资产、任务和复盘模板
05-角色与资产库/          未来资产存放接口
06-视频制作项目/          002闭环证据及未来正式项目
.agents/skills/           创意、逻辑审核、逐镜编译三个Skill
tools/                    轻量验证脚本
```

大型视频、音频、图片、模型、凭据和临时输出不应提交到 Git。
