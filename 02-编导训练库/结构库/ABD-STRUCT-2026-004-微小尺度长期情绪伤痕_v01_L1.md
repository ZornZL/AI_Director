# 微小尺度长期情绪伤痕结构

## 结构公式

```text
先抛出一个很小、很具体、可被量化的物理差异
→ 让一部分人物把它当成玩笑、调侃或无心评价
→ 让另一部分人物把它默默记住，但当下不反击
→ 用重复场景证明这个小差异持续影响其选择、关系或自我认同
→ 中段逐渐揭示：伤人的不是尺度本身，而是被轻慢、被比较、被忽略
→ 后段让原本开玩笑的人或旁观者意识到代价
→ 结尾回到同一个尺度、动作或空间距离
→ 让观众重新理解开头那句轻飘飘的话
```

## 核心机制

- **小尺度承载大情绪**：五厘米、一分钟、一句话、一步距离、一个座位、一条消息等微小单位，本身不重要，重要的是它被谁轻视、被谁记住。
- **双视角冲突**：一方觉得只是玩笑，另一方把它藏成羞耻、遗憾、错过或长期自我怀疑。
- **可见锚点替代抽象解释**：用身高线、门框刻度、座位距离、照片站位、未递出的东西等画面承载情绪，而不是只靠旁白说明。
- **重复不是重复信息**：同一个物理锚点每次出现都要改变含义，从玩笑、回避、逞强、错过，递进到理解或遗憾。
- **结尾回看完成反转**：结尾不靠新设定催泪，而是让观众发现开头的小事其实一直在影响人物。

## 情绪曲线

轻巧好奇 → 玩笑感 → 微妙不适 → 沉默积累 → 关系错位 → 真相回看 → 迟来的理解 → 遗憾留白。

## 使用要点

- 开头的量词必须非常具体，最好是能被画面看见的单位：厘米、分钟、一步、一个位置、一张照片、一句没说出口的话。
- 玩笑方不能写成单纯恶人，最好是无心、习惯性、群体性或自以为亲近，情绪才更真实。
- 被伤害方不要立刻解释全部痛苦，要用行为变化、回避动作、沉默停顿和后续选择逐步证明。
- 中段每次回到物理锚点时，都必须增加新的关系信息，不能只重复展示同一个画面。
- 结尾可以选择和解、错过、释怀或沉默，但必须让开头的“小事”获得新的重量。
- 如果用于品牌或产品故事，产品只能作为情绪锚点或关系动作的一部分，不能突然替代人物完成和解。

## 适用场景

- 青春、亲情、友情、职场、邻里、师生和老板IP观点故事。
- 需要用一个小细节切入大议题的短片，例如偏见、比较、亏欠、错过、尊重、成长或不被理解。
- 产品软植入中可用于“小物件承载长期关系”的变体，但产品必须从前段就参与动作和记忆。

## 禁用与谨慎场景

- 不适合把身体差异、外貌、身高、体重、疾病或贫穷当成笑点反复消费。
- 不得鼓励羞辱、霸凌、群体嘲笑或身体焦虑；如果出现玩笑，叙事立场必须清楚指向其代价。
- 不要固定复用“五厘米、身高差、校园玩笑、多年遗憾”的完整组合。
- 不要用字幕解释全部心理，否则会削弱“微小物理锚点”的画面力量。
- 精确刻度、数字、证件、手机消息等在 Seedance 中属于生成风险点，应优先用稳定视觉关系表达，不强求可读文字。

## 原创迁移边界

迁移时必须更换具体量词、人物关系、发生场景、伤害方式、回看方式和结尾动作。只能学习“小量词/小距离承载长期情绪”的机制，不得复刻原视频《有人把五厘米当成玩笑，有人把它藏在遗憾里》的具体人物、台词、场景组合、音乐、字幕版式和完整情节链。

## 可迁移方向

- 职场：一次会议座位边缘、一个没被介绍的名字，变成长期不被看见的伤口。
- 亲情：一分钟没等、一步没回头，成为多年后才理解的亏欠。
- 友情：照片里永远被裁掉的一点位置，揭示关系里长期的边缘感。
- 服务/产品：一件小物被反复忽略，最后成为关系修复的证据，但不得让产品替代人物情感。

## Seedance 2.0 验证重点

- 物理锚点要稳定可见，但不依赖精确可读文字。
- 人物视线、站位和距离要清楚，避免“谁在看谁、谁被排除”变得含糊。
- 群体玩笑场面要控制为克制生活化，避免生成明显霸凌、夸张表演或多人抢话。
- 反复出现的锚点建议用相似构图或相同空间关系建立记忆，但每次动作和人物反应要递进。
- 若涉及身高线、刻度、手机消息或照片文字，优先后期图文处理；Seedance 提示词中用“不可读的简洁标记”或“稳定距离关系”表达。
