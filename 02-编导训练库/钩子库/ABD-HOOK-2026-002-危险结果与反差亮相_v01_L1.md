# 危险结果与反差亮相钩子

## 钩子公式

```text
先展示高风险事件的混乱结果
→ 暂不解释过程和人物身份
→ 让疑似制造结果的角色正面亮相
→ 用与危险气氛相反的轻松反应完成反差
→ 片名或任务信息随后出现
```

## 为什么有效

- 结果先行制造“刚才发生了什么”的信息缺口。
- 危险环境快速交代类型，减少世界观说明。
- 外形、职业与反应之间的反差同时建立威胁感、幽默感和角色魅力。
- 片名后置，使观众先产生问题，再获得类型答案。

## 使用要点

- 危险结果必须清晰但不必依赖血腥细节。
- 角色反应应服务性格，不能只是随机卖萌。
- 后续应尽快证明角色确实具备处理该危险的能力。
- 可用表情屏、动作、道具或声音表达反差，不限定机器人角色。

## 原创边界

不复制泳池尸体、枪口、牛仔机器人笑脸和原片镜头顺序。迁移时更换危险类型、角色职业、反差方式和视觉环境。
