# 强情绪选择权递进结构

## 结构公式

```text
观众熟悉的尖锐质疑
→ 引导者不直接回答
→ 展示最低层的危险或损失
→ 主角提出合理反驳
→ 展示更现实、更强烈的第二层后果
→ 设置具体二选一
→ 主角亲自作出选择
→ 引导者将个案升华为价值主张
→ 主角用自己的话复述
→ 回到开场场景，以行动完成闭环
```

## 情绪曲线

现实刺痛 → 奇遇悬念 → 压迫 → 辛酸 → 动摇 → 醒悟 → 温暖。

## 使用要点

- 第一层案例负责证明基础价值，第二层案例负责放大现实代价和选择差异。
- 可以叠加受骗、亲人住院、受伤、收入和错失晋升等情绪证据，增强代入。
- 配角可以承担明确的情绪功能，快速推动主角认知变化。
- 结论应由原本反对的人说出，并在结尾转化为可见行动。
- 可采用直接、简化的因果链，保证短视频中心论点不被支线分散。

## 原创迁移边界

迁移时必须更换人物关系、案例内容、时空设定、视觉证据和结局行动，不得替换题材后复刻原故事链。

## 等级

L1：已从确认案例中提炼，尚未完成原创迁移验证。
