# 《告别》第一人称父辈短片案例卡

1. 样本编号：ABD-CASE-2026-005
2. 样本名称：《告别》（我们的父辈）
3. 原始素材类型：本地抖音参考视频，AI 亲情短片
4. 用户确认：2026-06-16，选择 B，只存案例
5. 样本等级：L1
6. 训练能力：第一人称 POV、人生节点串联、固定空间母题、代际告别回扣、强情绪亲情短片

## 用户满意的核心点

这条视频用“山路”作为一生告别的固定母题，把少年离家、恋人告别、外出谋生、婚礼缺席父母、孩子追问死亡、女儿出嫁、自己老去和下一代继续上路串成一条情绪线。它的有效点不在单个事件，而在每次告别都让主角身份变化：孩子、学生、恋人、工人、丈夫、父亲、外公，最后回到“父辈也曾这样送我走”。

## 不喜欢或需要避开

- 不复制原片完整事件链：山路、老师、恋人、工地、婚礼、父母墓、女儿出嫁、病榻、外孙回扣。
- 不复制具体台词，尤其是“天上的人会变成星星”“外公去天上了”“变成星星了”这一完整回扣。
- 不依赖 AI 直接生成准确文字道具；片中课本/作业本封面字形不稳定，涉及书本、证件、奖状、品牌文字时必须单独复核。
- 不把“哭、死亡、婚丧嫁娶”本身当作可学习点；真正可学习的是母题复现和身份递进。

## 可复用结构

```text
固定空间或固定路径开场
-> 主角第一次被送别
-> 同一空间/物件在不同人生节点反复出现
-> 每次告别推动主角身份升级
-> 中段用现实压力或死亡制造不可逆失去
-> 后段让主角从“被送别者”变成“送别人者”
-> 下一代重复当年的问题或动作
-> 用一句旧答案完成代际回扣
```

## 可复用钩子

以第一人称走在熟悉路径上的告别开场，让观众先进入“我正在离开家”的身体感；再通过多个年龄阶段的同一路径回返，让开场不只是空间，而成为整个人生循环的隐喻。

## 可迁移场景

- 亲情短片：父子、母女、祖孙、师生、养育与离别。
- 乡土题材：山路、渡口、车站、老屋门槛、村口桥、学校走廊等固定空间。
- 非乡土题材：医院走廊、地铁站台、工厂大门、训练场入口、公司电梯口等能反复承载“离开/归来”的位置。
- 人生阶段片：升学、入伍、进城、创业、成家、丧亲、送子女远行、退休或病榻告别。

## 原创边界

迁移时必须更换人物关系、关键事件、场景母题、陪伴物件、台词回扣和结尾动作。可以学习“固定空间母题 + 人生节点告别 + 代际反转”的方法，但不能照搬“山路 + 狗旺财 + 星星解释死亡 + 父辈一生告别”的完整组合。

## Seedance 2.0 验证重点

- 第一人称 POV 的手、物件、人物距离是否稳定，避免手部畸变和视角跳动。
- 同一人物跨年龄阶段的身份可识别度，尤其是成年、老年、病榻阶段。
- 狗、蓝布包、红色婚礼布包、作业本等关键物件的连续性。
- AI 文字道具必须作为高风险项处理，书本、署名、证件、标语不要依赖模型一次生成准确。
- 多阶段短片容易散，生成前必须锁定复现母题和回扣句，不然会变成事件拼贴。
