# 固定空间·重复对白·时间缺席闭环

## 结构公式

```text
建立一个可反复回到的固定空间或固定动作
→ 用早期日常建立被照顾、被等待的安全感
→ 用一次小冲突证明照顾者无条件站在主角一边
→ 跨越多个成长或人生节点
→ 主角每次以合理理由减少陪伴
→ 照顾者不断缩小要求，只保留最朴素的关心
→ 安排最后一次轻松互动，但不立即说明这是告别
→ 通过人物缺席或空间消失揭示时间代价
→ 让早期动作或台词在结尾重新出现
→ 同一句话获得相反意义，完成情绪闭环
```

## 核心机制

- **固定空间承载时间**：人物年龄和身份变化，空间关系保持稳定，让观众快速识别跨年代关系。
- **重复对白意义递变**：同一句生活化台词从日常、习惯、承诺逐渐变成遗憾或怀念。
- **合理疏离制造亏欠**：主角不是反派，每次缺席都有现实理由；照顾者越体谅，观众越替其感到委屈。
- **缺席而非争吵构成高潮**：高潮来自再也无法回应的人或已经消失的地方，不依赖激烈冲突。
- **回返完成翻转**：结尾不增加新口号，而是改变开场台词或动作的含义。

## 情绪曲线

亲切安全 → 被保护 → 成长疏离 → 期待缩小 → 短暂回暖 → 倒计时 → 缺席揭示 → 记忆回返 → 遗憾与怀念。

## 使用要点

- 固定空间、固定动作和固定台词至少保留两项，才能形成可识别的记忆锚点。
- 每个人生阶段只选择一个最能体现关系变化的事件，避免流水账。
- 主角的离开理由应真实合理，不要简单写成忘恩或恶意抛弃。
- 照顾者的要求应逐次缩小，让关系变化由行动和生活细节显现。
- 最后一次轻松互动要能在揭示后被观众重新理解，但不应提前用台词明示死亡。
- 缺席可以是离世、搬迁、失联、空间改建或身份交接，不必依赖死亡催泪。
- 片尾文字只能升格已有证据，不能替代前面的关系铺垫。

## 适用场景

- 亲情、师生、社区人物、职业传承和长期陪伴题材。
- 老字号、长期服务或陪伴型产品，但产品必须从前段开始参与重复行为。
- 需要用私人经验进入公共议题的人物致敬片。

## 禁用与谨慎场景

- 快节奏功能广告、即时促销和需要明确行动号召的内容，不适合以不可挽回的缺席收尾。
- 不得把真实公共人物虚构成亲属后混同为事实；必须标明致敬或象征性创作并核查事实。
- 不要固定套用“老人、夕阳、哭脸、竖排小作文”，这些只是表现形式，不是结构本身。
- 不要在悼念高潮突然植入产品，以免形成消费逝者或公共人物的观感。
- 儿童受伤、老人离世和拆迁等内容需单独检查平台合规与情绪伦理边界。

## 原创迁移边界

迁移时必须更换人物关系、固定空间、照顾行为、人生节点、缺席原因、公共议题指向和结尾动作。不得复刻“窗台奶奶—五毛钱—泡面—打架—求学—工作—结婚—拆迁—童年回返”的完整链条，也不得照搬原对白、信件、音乐、角色造型和版式。
