# 答案型文案说话关系表

## 等级

L1：由《五厘米》参考文案分析和用户确认规则提炼。用于答案型群像文案、街采式命题短片、亲情/友情/遗憾类短片进入正片文案前的强制检查。尚未完成新项目 L2 验证。

## 核心规则

所有答案型文案在进入正片文案前，必须先过以下链路：

```text
角色身份
→ 说话对象
→ 缺席关系
→ 未说出口的代价
→ 画面锚点
→ 人生哲理
```

如果一个答案只能说明“他说了什么”，不能说明“他说给谁、为什么难、没说会怎样”，不得进入正片文案。

## 字段定义

### 1. 角色身份

说明这个人是谁，以及他处在什么生活阶段、职业、家庭或关系压力里。

合格示例：

- 30岁北漂青年，独居，刚挂掉妈妈电话，习惯报喜不报忧。
- 45岁医生，刚要向家属说明坏消息，自己也是父亲。
- 78岁小卖部老人，反复播放孙子留下的一条语音。

不合格示例：

- 一个青年。
- 一个老人。
- 一个普通人。

### 2. 说话对象

说明这句话真正说给谁听。对象可以是在场人物、缺席人物、过去的自己、逝去的人、某段关系或命运。

合格示例：

- 表面回答采访者，实际说给妈妈。
- 表面看着镜头，实际说给没能救下的人。
- 表面是自嘲，实际说给被生活磨损的自己。

### 3. 缺席关系

说明关键对象是否缺席，以及缺席如何制造情绪。

合格示例：

- 妈妈不在场，只在通话记录里。
- 老朋友不在场，只剩半局棋。
- 孙子不在场，只剩一条不到一分钟的语音。

没有缺席也可以，但必须说明在场关系如何产生压力，例如父子面对面却说不出口。

### 4. 未说出口的代价

说明这句话如果没有说、没有做、没有等，会留下什么后果或余味。

合格示例：

- 没说“我想你”，想家被伪装成“我挺好的”。
- 没问“累不累”，亲情继续变成沉默。
- 没多等一分钟，一个人可能错过最后一班回家的车。

### 5. 画面锚点

说明观众不用听解释，也能看见情绪的物件、动作或空间。

合格示例：

- 手机停在“妈”的通话记录上。
- 门快关上，父亲拎着菜站在电梯外。
- 木桌上的旧手机、播放键、门口风铃。

### 6. 人生哲理

哲理不能直接塞进角色台词，必须由角色处境推出。写作时用于创作者判断，不一定进入正片。

合格示例：

- 成年人的懂事，常常把想念藏成“我挺好的”。
- 亲情变远，往往不是不爱了，是问候来得太少。
- 最想借的时间，不是为了重来，是为了再等一个爱的人。

## 强制表格模板

```text
角色身份：
说话对象：
缺席关系：
未说出口的代价：
画面锚点：
人生哲理：
入片答案：
```

## 入片硬门槛

答案型文案进入 G2 正片文案前必须满足：

1. 每个入片答案都填完说话关系表。
2. 至少 70% 的答案有明确的“缺席关系”或“说不出口的对象”。
3. 每个答案都有一个可见画面锚点。
4. 角色台词不得直接说哲理；哲理留给观众自己悟。
5. 情绪高潮答案必须同时具备缺席关系、未说出口的代价和强画面锚点。
6. 入片答案必须精简。优先短句、半句话、停顿后的补句或具体动作后的低声表达；不得用长台词解释人生道理。

## 精简深意台词检查

每句入片答案进入正片前必须再问：

1. 能否删掉解释词，只留下关系和代价？
2. 这句话是否真的说给某个具体对象，而不是说给观众？
3. 观众能否从画面锚点和上下文悟到更深含义？
4. 如果把“人生、亲情、遗憾、珍惜、成长”等抽象词删掉，台词是否更真实？
5. 这句话是否短到适合短视频角色自然说出口？

推荐形式：

```text
短句 + 停顿 + 具体对象
半句话 + 没说出口的后果
具体动作后的一句低声表达
```

## 对《借你一分钟》的应用

问题：“如果能借你一分钟，你想把它放在哪里？”

示例：

```text
角色身份：30岁北漂青年，独居，刚挂掉妈妈电话，习惯报喜不报忧。
说话对象：表面回答采访者，实际说给妈妈。
缺席关系：妈妈不在场，只在通话记录里。
未说出口的代价：想家被伪装成“我挺好的”。
画面锚点：出租屋、没拆的泡面、手机停在“妈”的通话记录上。
人生哲理：成年人的懂事，常常把想念藏成报喜不报忧。
入片答案：放到挂电话前吧，我想说，我也想你。
```

## 对《少一句话》的应用

问题：“如果能少一句话，你最想少掉哪一句？”

示例：

```text
角色身份：18岁高三生，早上和妈妈吵架后出门。
说话对象：表面回答采访者，实际说给门后的妈妈。
缺席关系：妈妈在门后，但两个人隔着一扇门和一口气。
未说出口的代价：一句气话可能变成一整天的后悔。
画面锚点：楼道门把手、门里碗筷声、背上的书包。
人生哲理：很多关系不是被大事伤到，而是被一句没收回的话推远。
入片答案：少掉关门前那句“别管我”。
```

## 禁用

- 不得让角色直接讲“人生就是……”这类作者观点。
- 不得只用职业标签代替角色身份。
- 不得只写“亲情、遗憾、勇气”等抽象命题，没有具体对象。
- 不得把所有答案都说给采访者或观众；必须找到真正的情绪对象。
