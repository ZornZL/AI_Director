## 用户满意的核心点

- 学校门口钩子、社区时间错位、雨夜牵手、睡前袖口等情绪关系镜头成立。
- 黄精果作为“女儿小时候爱吃的零食”和妈妈记忆中的照顾物件，具备情感信物功能，不是硬广讲解。
- 多个低运动母女关系戏可以通过 8-12 秒合并镜头完成。
- 真实生成和剪辑证明“简单文案 → 分镜提示词 → 网页端生成 → 剪辑成片 → 复盘优化”的闭环可行。

## 不喜欢或需要避开

- 角色对空气说话、对前方说话、对镜头说本应说给对方的话。
- 妈妈和女儿不同年龄阶段人物混淆，尤其大学送别段妈妈女儿视觉差不够。
- 手机朝向观众导致人物像对镜头聊天，而不是对手机聊天对象说话。
- 葱花、黄精果、碗面、纸巾等小物件没有明确终态。
- 产品递接方向反转，或产品突然从无到有出现。
- 角色额外说出提示词外台词，或音色重复、音色归属混乱。
- 情绪镜头中人物站位太远，母女关系像陌生人帮助老人。

## 训练目标

训练 AiBianDao 的核心生产链：

```text
锁定文案
→ 资产/声音/画面边界
→ Seedance 2.0 分镜提示词
→ 实际生成
→ 用户筛选和剪辑
→ 成功/失败复盘
→ 编译器规则升级
```

本样本重点训练：视线对象、说话对象、道具递接、手机视角、小物件终态、多年龄角色隔离、额外台词控制、产品情感信物植入和合并镜头失败降级。

## 可复用结构

```text
现实照护钩子
→ 童年母题台词来源
→ 成长阶段不断推开妈妈
→ 成年后错过妈妈
→ 妈妈失智但仍记得女儿细节
→ 女儿开始反向照顾
→ 母题台词反向回扣
→ 床边陪护与手部留白收束
```

该结构适用于亲情、照护、记忆退化、角色位置反转、迟到理解类短片。

## 可复用钩子

- 认知错位钩子：白发妈妈拿着给“小女儿”的食物，问成年女儿“你认识我女儿吗？”
- 情感信物钩子：产品不讲功能，只作为妈妈一直记得女儿偏好的物件。
- 母题台词钩子：“车多，不能松手”从妈妈保护女儿，反转为女儿保护妈妈。

## 可迁移产品或场景

- 产品适合：零食、常吃小物、随身物件、家庭记忆物、非强功效表达产品。
- 场景适合：学校门口、路口、办公室、厨房、社区、餐桌、卧室。
- 不适合：需要强功效证明、复杂包装露出、精确长文字持续展示的硬广产品。

## 原创边界

- 可迁移的是“照护关系反转 + 情绪信物 + 母题台词回扣”机制。
- 不应照搬本片人物组合、完整台词链、全部场景顺序和黄精果具体设定。
- 产品植入必须保持亲情记忆物件边界，不延伸为儿童功效、疾病治疗、滋补承诺或保健承诺。

## Seedance 2.0 生产规则

1. 说话镜头必须写清“说给谁听”：视线对象、身体朝向、对方已停稳或已接住后才开口。
2. 情绪关系镜头必须写清人物距离，避免母女看起来像路人。
3. 手机镜头必须区分“观众读字”和“角色使用手机”；角色语音优先手机朝向角色或过肩/第一人称视角。
4. 小物件动作必须有落点和终态：葱花放到小碟、产品在谁手中、碗面是否无葱、纸巾擦掉什么。
5. 产品情感信物镜头必须写清所有权方向：谁递出、谁接稳、谁松手、尾帧谁持有。
6. 多年龄阶段人物必须建立阶段差异，不能只靠同一角色资产抽卡。
7. 对白必须使用白名单，防止 Seedance 增加、重复或改写台词。
8. 同类关键失败两次后，优先拆镜或补首帧，不继续在原合并镜头堆描述。
9. Seedance 分镜视频默认不生成背景音乐、背景声铺底、歌词、字幕、片名、黑屏字卡和片尾文案；但必须保留人物说话声音，以及人物场景中真实发生且应当发出的剧情内声音。字幕、背景音乐和声音包装在后期发布阶段添加。

## 声音与音色经验

本片未预先建立严格音色资产，而是通过“70岁妈妈、35岁成年女儿”等文本描述抽卡生成。结果说明：简单短片可以先用年龄身份描述生成声音，但若要跨镜稳定复用角色声音，应建立“音色资产”或“声音圣经”。

建议后续将每个主要角色登记：

- 基础音色：年龄感、声线高低、语速、气息、口音、情绪底色。
- 情绪变体：生气、开心、懊恼、疲惫、温柔、迷茫、哽咽等只作为基础音色上的表演变化，不作为另一个新声音。
- 禁止变化：不得突然年轻化、换性别、换口音、换成旁白腔、两个角色音色混用。

声音边界修正：音色资产只服务于必要对白、口型同步和剧情内事件声，不等于要求 Seedance 同时生成背景音乐或字幕。为了降低抽卡率，后续分镜提示词应明确“本镜保留人物说话和必要剧情内声音，不生成背景音乐、不生成背景声铺底、不生成字幕，后期添加字幕、背景音乐和声音包装”。

## 首帧/关键帧经验

本片所有分镜视频均使用资产引用出视频，没有使用生产型首帧、首尾帧或关键帧。结果证明：在核心资产引用充足、动作单一、低运动关系戏、文字较短、产品不要求商业级标准露出时，可以通过纯资产引用 + 明确提示词完成可用成片。

但本片也证明以下镜头仍建议补首帧或拆镜：

- 产品标准露出。
- 精确中文或手机界面。
- 多年龄角色同框且容易混淆。
- 产品递接、葱花、手部、袖口等动作终态必须严格验收。
- 需要继承上一镜位置、朝向、手部或道具状态的镜头。
