## 保守拆分版

保守拆分版沿用 G2 的 13 镜结构。优点是失败成本低，每个镜头只承担一个主要信息点；缺点是网页端操作次数更多。

| 镜头 | 建议 | 理由 |
|---|---|---|
| SHOT_01 | 保留 | 首个门后空间转换，必须单独测试 |
| SHOT_02 | 保留 | 奇幻回现实，旁白和人物误动嘴需单独验收 |
| SHOT_03 | 保留 | 手机语音、按掉语音和“等忙完”需要独立控制 |
| SHOT_04 | 保留 | 洗手间到院子的空间转换与母亲抱被子 |
| SHOT_05 | 保留 | 电梯到饭桌的空间融合风险高 |
| SHOT_06 | 可拆 | 蒙太奇信息密度高，失败时拆成4个短插入 |
| SHOT_07 | 保留 | 修旧门把手和最重台词，A级风险 |
| SHOT_08 | 可拆 | 双人多句对白，口型失败时拆成正反打 |
| SHOT_09 | 保留 | 挡门、关机、台词是高潮动作，A级风险 |
| SHOT_10 | 保留 | 洗手和母亲一句台词，低运动情绪镜 |
| SHOT_11 | 可拆 | 双人三句对白，口型失败时拆成陈远近景、母亲反应、陈远近景 |
| SHOT_12 | 保留 | 旁白直出且人物不误动嘴 |
| SHOT_13 | 保留 | 片尾静物留白，无字验收 |

## 合并生产版

合并生产版适合用户想减少操作次数，但应在保守版至少测试通过关键风险后再用。

| 合并段 | 包含镜头 | 建议时长 | 合并条件 | 风险 |
|---|---|---:|---|---|
| MERGE_01 | SHOT_01 + SHOT_02 | 8-10秒 | 门后空间转换已稳定，旁白不误动嘴 | 空间转换和旁白同时失败时难定位 |
| MERGE_02 | SHOT_03 | 12秒 | 不建议合并 | 手机语音和台词归属需独立控制 |
| MERGE_03 | SHOT_04 | 12秒 | 不建议合并 | 洗手间/院子转换独立验收 |
| MERGE_04 | SHOT_05 | 12秒 | 不建议合并 | 电梯/饭桌空间融合独立验收 |
| MERGE_05 | SHOT_06 | 12秒 | 可直接做蒙太奇 | 信息密度高，失败时拆分 |
| MERGE_06 | SHOT_07 + SHOT_08 | 15秒 | 只有在双人对白和门把手动作稳定后再合并 | 手部道具和多句对白叠加 |
| MERGE_07 | SHOT_09 + SHOT_10 | 15秒 | 挡门关机和洗手动作都测试通过后可合并 | 动作转场可能乱 |
| MERGE_08 | SHOT_11 | 12秒 | 不建议合并 | 情绪落点需要清晰口型 |
| MERGE_09 | SHOT_12 + SHOT_13 | 12-14秒 | 旁白不误动嘴，片尾无字稳定 | 片尾可能误生成文字 |

## 当前推荐

先使用保守拆分版生成 SHOT_01、SHOT_07、SHOT_09、SHOT_12 四个高风险测试镜头。四个通过后，再按 13 镜完整生成；不要一开始使用合并生产版。
