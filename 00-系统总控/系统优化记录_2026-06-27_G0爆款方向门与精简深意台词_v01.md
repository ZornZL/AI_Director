# 系统优化记录：G0爆款方向门与精简深意台词 v01

日期：2026-06-27
状态：已写入当前生产规则；这是工作流和文案质量门升级，不是单条参考视频训练入库。

## 触发问题

用户指出：当前系统输出的视频文案常常比较普通，不符合抖音爆款逻辑。理想文案应具备：

- 强钩子，先让用户停留。
- 中间情绪递进，而不是并列答案或散文化抒情。
- 角色说话精简，但每句话隐含深刻人生哲理。
- 末尾深化主题并引人深思。

## 原因判断

当前系统已经具备较强的 Seedance 生产编译能力，但前端创意筛选仍偏弱：

1. 容易过早进入完整文案，导致方向没有先经过停留能力筛选。
2. “停留-递进-故事-呼应”更多被用作写完后的验收表，而不是写之前的创作发动机。
3. 角色台词容易直接讲道理，缺少具体说话对象、未说出口的代价和画面承载。
4. 中段容易变成答案合集，段落顺序可以随意调换，说明没有真实递进。
5. Seedance可制作性会反向压低文案锋利度，导致文案稳但普通。

## 本次新增强制规则

### 1. G0爆款方向门

从主题、命题、产品、参考片或空白需求开始创作文案时，完整文案前必须先过G0。

G0是新增人工确认门。用户确认方向、8秒开场、情绪阶梯和结尾回扣后，才进入完整文案。

每个候选方向必须包含：

- 一句话命题。
- 前3-8秒钩子。
- 第8秒和第20秒停留理由。
- 人物关系和核心冲突。
- 3-5层情绪阶梯。
- 最重的一句话。
- 结尾回扣动作、物件、台词、问题或关系。
- 产品/品牌进入方式。
- Seedance生产风险和爆款风险。

### 2. 8秒开场测试

推荐方向进入完整脚本前，至少比较三种开场：

- 认知错位型。
- 轻问题重答案型。
- 反常动作/关系悬念型。

只选出最强开场或明确沿用用户指定开场后，才继续完整文案。

### 3. 情绪阶梯表

中段不能是并列案例，必须形成递进。推荐阶梯：

```text
轻冲突 -> 现实处境 -> 关系代价 -> 不可逆遗憾 -> 主题回扣
```

如果段落可以随意调换顺序，视为没有通过爆款叙事验收。

### 4. 精简深意台词

角色关键台词必须做到：

- 短：一句话只承载一个动作、对象或未说出口的情绪。
- 准：必须知道说给谁。
- 藏：人生哲理藏在关系、代价、物件和停顿里，不直接说破。
- 重：这句话不说、晚说或说错会有后果。
- 可演：画面中有动作、眼神、物件或空间承接。

禁止让角色直接说“人生就是”“我们应该珍惜”“亲情最伟大”等作者口号。

## 后续执行口径

以后系统不能把“写完整文案”当成创意任务的第一步。必须先判断方向是否有停留能力、递进能力、关系代价和结尾回扣。角色台词宁可少，也要让观众从关系和画面里悟出更深含义。

## 仍需观察

该规则目前为系统工作流升级，尚未完成多个真实项目验证。后续至少需要用2-3个不同题材项目测试：

1. 是否显著降低普通散文化文案。
2. 是否提高开头停留强度。
3. 是否避免台词过度抽象或过度解释。
4. 是否在不牺牲Seedance可制作性的情况下保留文案锋利度。
