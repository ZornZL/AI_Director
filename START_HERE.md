# START_HERE

AI_Director is a beginner-friendly Seedance 2.0 directing and shot-compilation system.

Read first:

- `从这里开始.md`
- `00-系统总控/总索引.md`
- `00-系统总控/当前系统能力边界.md`

Main chain:

```text
theme or competition brief
-> three creative directions
-> user choice
-> complete script
-> logic review and explicit lock
-> director shots
-> assets/keyframes
-> independent 4-15 second Seedance tasks
-> manual generation, review, and retry
```

The only validated full-loop case is `ABD-PROD-2026-002-houlai-mama`. Storyboards are optional previews, not a validated mandatory gate.
