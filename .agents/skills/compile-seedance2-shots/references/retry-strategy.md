# 失败重试策略

## 重试顺序

1. 保持剧情和台词不变，删除无关装饰描述。
2. 固定机位或减小运镜。
3. 每镜只保留一个主动作。
4. 把多人对白拆成正反打。
5. 把产品展示与人物复杂表演拆开。
6. 增加生产型首帧或动作参考。
7. 缩短任务并在语义停顿处拆镜。
8. 仍失败时更换表现方案，但必须保留原始信息功能。

同一合并镜头连续两次出现同类关键失败时，不继续在原长镜头里叠加更多禁止项。优先拆镜或补首帧：动作失败拆成准备、执行、结果；手机/便签/包装文字失败拆成文字插入；两人对白或音色失败拆成单人反应/正反打；产品递接方向失败拆成产品起点、递出、接稳三个可验收状态。

## 常见失败

- 人物漂移：减少角色数量，强化身份资产，降低转身和遮挡。
- 口型失败：缩短台词，明确唯一说话者，减少动作和运镜。
- 产品变形：独立低运动特写，减少手部遮挡和旋转。
- 动作混乱：拆成准备、执行、结果三个镜头。
- 空间跳变：固定背景锚点，继承上一镜尾帧。
- 情绪不明显：改写为可见的眼神、呼吸、肩颈和手部反应。
- 对空气说话或看错方向：增加说话对象、视线目标、身体朝向、人物距离和“对方停稳后才开口”；禁止看镜头或对前方说话。若仍失败，拆成说话者近景和听者反应镜头。
- 手机视角错误：先判断手机是给观众读字还是给角色使用。角色语音时写“手机屏朝向角色/过肩视角/第一人称视角”；观众读字时拆成手机屏插入，不让人物对镜头说话。
- 道具方向反转：写清道具所有权、移动方向、接收方、松手、尾帧持有者。产品信物不得只写“递给她”，必须写“妈妈递出，女儿接稳，妈妈松手，尾帧产品在女儿手中”。
- 小物件没有终态：指定容器或落点、重复次数和完成画面，例如葱花放到小白瓷碟里，碗面表面不再有明显葱花，筷子停在碗沿旁。
- 额外台词或声音重复：缩短时长，减少对白，只保留台词白名单；必要时拆成单句对白镜头或改画外音，不能只追加“不要乱说话”。
- 自动字幕、片名、片尾字卡、背景音乐、歌词或环境声铺底：删除所有字幕和音乐生成要求，在 prompt 中明确“本镜保留人物说话和必要剧情内声音，不生成背景音乐、不生成背景声铺底、不生成歌词、不生成字幕，后期再添加字幕、背景音乐和声音包装”；如果仍出现，缩短时长并降低对白/动作复杂度。
- 多年龄角色混淆：更换或补充阶段专属角色资产，强化年龄差、发型、服装和动作职责。若仍混淆，避免同镜长时间双人同框，拆成妈妈近景、女儿近景和关系反应。
- 剪辑手柄不足：减少剧情动作占用时长，在提示词和逐秒时间轴中明确开头稳定0.2-0.3秒、结尾稳定0.3-0.4秒；长对白、合并镜头或情绪承接镜头可把结尾稳定尾帧提高到0.5秒。插入镜头、产品特写、手机/便签文字、片尾留白和手部静态特写不要用0.8-1.0秒视频尾帧换稳定，优先生成清晰短尾帧，再在剪辑中定格或延长图片层。动作接剪镜头只保留0.2-0.3秒并标注接剪用途。

每次重试只改变一到两个变量，并记录失败类型、修改内容和结果，避免无法判断哪项修改有效。
