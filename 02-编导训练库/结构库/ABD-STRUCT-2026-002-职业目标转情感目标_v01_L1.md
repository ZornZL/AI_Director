# 职业目标转情感目标结构

## 结构公式

```text
危险结果或任务现场先行
→ 建立角色职业、规则与高能力
→ 完整兑现一次原类型爽点
→ 高潮后降速，暴露角色未满足的内在需求
→ 意外人物或关系目标出现
→ 主角主动改变原有注意力与行动目标
→ 用低对白仪式蒙太奇建立关系
→ 原有危机回归并检验新选择
→ 主角把原职业能力用于保护新关系
→ 双方共同进入不可逆的新阶段
```

## 核心机制

- 先兑现开场承诺，再换目标；不能用情感线逃避前半段承诺。
- 高潮后的低潮负责暴露角色“能做什么”之外的“缺什么”。
- 新关系必须改变后续行动，而非只作为结尾奖励。
- 原有能力在终局获得新用途，完成人物身份变化。
- 结尾通过空间、身份或关系状态的不可逆变化证明选择已经发生。

## 情绪曲线

威胁好奇 → 能力爽感 → 高潮释放 → 空虚或孤独 → 意外吸引 → 甜蜜建立 → 危机检验 → 主动选择 → 温暖余韵。

## 适用场景

- 动作、职业、冒险、科幻、灾难、游戏、角色 IP 与音乐短片。
- 需要从功能卖点转向情感价值的品牌故事。
- 对白较少、依靠视觉符号跨语言传播的内容。

## 禁用条件

- 前半段任务没有完成基本兑现，观众会认为影片突然跑题。
- 角色没有先显露内在缺口，关系转向会像强行恋爱。
- 新人物不影响选择和结局，只承担外形展示。
- 产品或品牌与任务、关系、逃离方案均无因果联系。

## 原创迁移边界

必须更换职业、世界规则、关系对象、视觉符号、危机类型、共同经历和结局行动。不得复刻“机器人牛仔清丧尸、骑鸵鸟、遇红裙机器人、乘飞船离开地球”的链条。
