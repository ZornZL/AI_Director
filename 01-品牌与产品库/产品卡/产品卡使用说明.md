# 产品卡使用说明

## 产品卡是什么

产品卡是AI编导读取单个产品时使用的“事实说明书和创作边界”。它不是销售文案，也不是功效说明。

产品卡主要解决五个问题：

1. 这个产品在画面里到底是什么，包装和名称不能生成错。
2. 哪些信息有包装、标签或公司资料依据，可以公开表达。
3. 哪些功效、疾病和绝对化表述不能出现。
4. 产品适合进入哪些人物、场景和故事，而不显得像硬广告。
5. Seedance 2.0生成时需要准备哪些产品参考图和验收标准。

## 使用规则

- 每次涉及产品，只读取本次产品对应的卡片，不要一次加载全部产品。
- 标注“待公司确认”的字段不得由AI自行补全，更不得写入正式视频台词。
- 传统原料的民间认知、古籍记载或网络说法，不等于该产品可以宣称的功效。
- 产品卡中的“故事母题”仅用于创意联想，不能变成功效暗示。
- 正式制作前，必须用真实包装、标签、说明书和公司审核材料更新至V1。

## V0升级为V1所需资料

1. 产品六面包装高清图和内容物照片。
2. 正式品名、食品类别、净含量、规格和食用方法。
3. 完整配料表、营养成分表、执行标准和生产信息。
4. 公司审核通过的可说卖点、证据材料和禁用词。
5. 官方价格区间、销售渠道和目标消费场景。
